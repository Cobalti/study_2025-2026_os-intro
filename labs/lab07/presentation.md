## Структура файловой системы Linux

Файловая система Linux имеет иерархическую структуру, начинающуюся с корневого каталога `/`.

| Каталог | Назначение |
|---------|------------|
| `/` | Корневой каталог |
| `/bin` | Основные пользовательские команды |
| `/etc` | Конфигурационные файлы |
| `/home` | Домашние каталоги пользователей |
| `/usr` | Пользовательские программы |
| `/var` | Переменные данные (логи, кэши) |

## Команды для просмотра файлов

| Команда | Назначение |
|---------|------------|
| `cat` | Вывод всего содержимого файла |
| `less` | Постраничный просмотр с навигацией |
| `head` | Вывод первых строк файла (по умолчанию 10) |
| `tail` | Вывод последних строк файла (по умолчанию 10) |

---

## Управление навигацией в less

| Клавиша | Действие |
|---------|----------|
| `Space` | Переход к следующей странице |
| `ENTER` | Движение вперёд на одну строку |
| `b` | Возврат на предыдущую страницу |
| `h` | Вызов справки |
| `q` | Выход из режима просмотра |

---

## Команды для управления файлами

| Команда | Назначение | Опции |
|---------|------------|-------|
| `cp` | Копирование файлов и каталогов | `-r` — рекурсивно, `-i` — запрос подтверждения |
| `mv` | Перемещение и переименование | `-i` — запрос подтверждения |
| `mkdir` | Создание каталога | `-p` — создание промежуточных каталогов |
| `rm` | Удаление файлов и каталогов | `-r` — рекурсивно, `-f` — без подтверждения |

## Права доступа

| Право | Обозначение | Для файла | Для каталога |
|-------|-------------|-----------|--------------|
| Чтение | `r` | Просмотр и копирование | Просмотр списка файлов |
| Запись | `w` | Изменение и переименование | Создание и удаление файлов |
| Выполнение | `x` | Запуск программы | Доступ в каталог |

|----------|--------------|------------|
| 111 | 7 | rwx |
| 110 | 6 | rw- |
| 101 | 5 | r-x |
| 100 | 4 | r-- |
| 011 | 3 | -wx |
| 010 | 2 | -w- |
| 001 | 1 | --x |
| 000 | 0 | --- |

## Изменение прав доступа

Команда `chmod` позволяет изменять права доступа:

**Символьный режим:**
- `chmod u+x файл` — добавить выполнение для владельца
- `chmod g-w файл` — убрать запись для группы
- `chmod o=r файл` — установить только чтение для остальных

**Цифровой режим:**
- `chmod 755 файл` — rwxr-xr-x
- `chmod 644 файл` — rw-r--r--
- `chmod 400 файл` — rwx------
