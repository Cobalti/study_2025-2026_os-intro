# Цель работы

Ознакомление с файловой системой Linux, её структурой, именами и содержанием каталогов. Приобретение практических навыков по применению команд для работы с файлами и каталогами, по управлению процессами, по проверке использования диска и обслуживанию файловой системы.

**Задачи:**
- Изучить основные команды для работы с файлами и каталогами
- Освоить операции копирования, перемещения и переименования файлов
- Изучить систему прав доступа и научиться изменять их
- Приобрести навыки анализа файловой системы
- Научиться просматривать содержимое файлов различными способами

# Вывод

В ходе выполнения лабораторной работы были достигнуты следующие результаты:

- Изучена файловая система Linux, её структура и основные каталоги
- Освоены команды для работы с файлами и каталогами:
  - Просмотр содержимого файлов (`cat`, `less`, `head`, `tail`)
  - Копирование файлов и каталогов (`cp`)
  - Перемещение и переименование (`mv`)
- Изучена система прав доступа к файлам и каталогам
- Освоено изменение прав доступа с помощью команды `chmod`
- Выполнены практические упражнения по созданию иерархии каталогов и файлов
- Изучены команды для анализа файловой системы: `mount`, `df`, `fsck`

**Практическая значимость:** полученные навыки позволяют эффективно работать с файловой системой Linux, управлять правами доступа, организовывать структуру каталогов и выполнять базовые операции с файлами, что является необходимым для дальнейшей работы в операционной системе Linux.
