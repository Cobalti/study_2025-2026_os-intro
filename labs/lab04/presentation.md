# Теоретические сведения

## Системы контроля версий

**VCS** — система для отслеживания изменений в файлах проекта.

| Тип | Примеры | Особенность |
|-----|---------|-------------|
| Централизованные | SVN, CVS | Единый сервер |
| Распределённые | Git, Mercurial | Полная копия у каждого |

## Основные команды Git

```bash
git init          # инициализация
git add .         # добавление файлов
git commit -m ""  # сохранение изменений
git push          # отправка на сервер
git pull          # получение обновлений
```
