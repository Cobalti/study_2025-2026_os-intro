# Теоретическое введение

## GitHub Pages

**GitHub Pages** — это сервис статического хостинга сайтов, который позволяет размещать веб-страницы непосредственно из репозитория GitHub.

### Возможности GitHub Pages

| Функция | Описание |
|---------|----------|
| **Статические сайты** | Хостинг HTML, CSS, JavaScript файлов |
| **Автоматическая сборка** | Поддержка Jekyll и других генераторов |
| **HTTPS** | Автоматическое SSL-шифрование |
| **Custom domains** | Поддержка собственных доменов |

## Hugo

**Hugo** — это генератор статических сайтов, написанный на Go. Отличается высокой скоростью работы и простотой использования.

### Преимущества Hugo

| Характеристика | Описание |
|----------------|----------|
| **Скорость** | Очень быстрая генерация сайтов |
| **Шаблоны** | Большое количество готовых тем |
| **Простота** | Лёгкая установка и настройка |
| **Markdown** | Поддержка Markdown для контента |

## Git

**Git** — распределённая система контроля версий для отслеживания изменений в исходном коде.

### Основные команды Git

| Команда | Назначение |
|---------|------------|
| `git clone` | Клонирование репозитория |
| `git add` | Добавление файлов в индекс |
| `git commit` | Фиксация изменений |
| `git push` | Отправка изменений на сервер |
| `git pull` | Получение изменений с сервера |
